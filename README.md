### SpringBoot + security + jwt 安全校验（未包含网关）

1. 用户登录，发放jwt令牌token以及刷新token
2. 自定义角色
3. 自定义接口
4. 自定义接口与角色的关系（授权角色）
5. 自定义接口开放以及定义角色权限
6. 用户发布论坛帖子，后续增加管理员审核功能
7. 自定义敏感词汇检测
